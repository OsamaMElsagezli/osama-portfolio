AI App for Plant Disease Detection

A mobile application for plant disease detection using AI and computer vision concepts to solve a practical real-world agricultural problem.

Category: Mobile App Development

Tools: Flutter, Dart, TensorFlow, Computer Vision

Role: Mobile app development and AI integration for plant disease detection

Project Features

  • AI-powered plant disease detection using deep learning techniques
  • Real-time disease identification from plant leaf images
  • Mobile application (Plantly) for on-the-go disease diagnosis
  • Image capture and instant disease prediction workflow
  • Support for multiple crop species and disease categories
  • Integration of PlantVillage and PlantDoc datasets for improved robustness
  • High-accuracy disease classification with optimized model selection
  • User-friendly Flutter interface designed for farmers and agricultural use

Technical Highlights

  • Developed using Flutter and Dart for cross-platform mobile deployment
  • Implemented deep learning models using PyTorch
  • Compared EfficientNetB3, ResNet50, VGG16, and MobileNetV2 architectures
  • Applied transfer learning with pretrained CNN models
  • Utilized Albumentations for image preprocessing and data augmentation
  • Performed image normalization, resizing, and dataset balancing techniques
  • Used GPU-accelerated mixed-precision training for improved performance
  • Achieved up to 99.75% test accuracy with EfficientNetB3
  • Evaluated models using Accuracy, Precision, Recall, F1-Score, and Confusion Matrices
  • Designed for future real-time deployment and smart agriculture applications
